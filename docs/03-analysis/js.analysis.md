- **[Med] login.html / signup.html — 공통 푸터 미적용**: 두 페이지만 `<footer id="footer">` 요소가 없어 공통 푸터가 렌더되지 않음(헤더는 21개 전부 주입). 인증 페이지 미니멀 의도로 보이나 CLAUDE.md B-2 "모든 페이지가 같은 헤더/푸터"와 어긋남 → 의도라면 B-2에 예외 명시, 아니면 푸터 껍데기 추가.
- **[Low] login/signup의 head/script 링크가 한 줄로 압축**: 기능 문제 없음, 편집 일관성만 저하.
- **[Low] `.pv-card__label`이 일부 카드에만 존재**: B-6 문서의 표준 구조(모든 카드에 label)와 실제(라벨은 선택 요소)가 다름 → 문서 갱신 권장.
- **[Low] products.html에 가격이 정적 텍스트로 중복**: 단일 출처는 app.js `PRODUCTS`이며 현재 값은 완전 일치(248,000/312,000/276,000/389,000/24,000/55,000). 가격 변경 시 두 곳 동시 수정 필요(드리프트 위험) → B-5 절차에 명기 권장.
- **[Low] `template/` 폴더가 B-1 폴더 구조에 미기재**: 원본 Moderna 보관용(우리 페이지는 CDN 폰트 잔재 없음). 제출본 포함 여부 결정 필요.
- **[Low] 페이지 인라인 스타일에 하드코딩 hex 일부**: custom.css는 토큰 활용 우수(var 166회), 인라인은 B-2 #3상 허용 범위.
