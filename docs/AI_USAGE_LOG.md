### Vivienne Westwood × 이천쌀 프로젝트 — 발표·제출용

> 평가 안내에 따라, AI 도구 사용 여부와 **본인이 직접 판단·수정한 부분**을 기록합니다.

## 1. AI 도구 사용 여부
- **사용함.** 사용한 도구: 대화형 AI(기획 정리·문서화·코드 생성 보조), 이미지 생성/편집 AI(콜라보 비주얼·로고 모핑 이미지 제작 보조).
- 사용 목적: 기획을 빠르게 문서로 정리하고, 반복 작업(코드 골격, 자료 검색)을 보조받기 위함.
- **사용하지 않은(=직접 한) 핵심 영역**: 콜라보 컨셉 기획, 와이어프레임 디자인, 사이트 구조 설계, 페이지별 인터랙션 결정.

## 2. AI를 사용한 부분
- 인터뷰 형식으로 기획 의도를 질문받고, 결정 사항을 **명세서(PROJECT_SPEC.md)로 정리**하는 작업
- 이천 3대 축제 **공식 홈페이지 URL 검색** (도자기/산수유/쌀문화)
- Bootstrap(Moderna) 템플릿 분석 및 **"어떤 컴포넌트를 어디에 활용/수정할지" 매핑표** 초안 작성
- **Figma MCP 연동**: 내가 디자인한 Figma 프레임/노드 링크를 주면 AI가 디자인·에셋을 직접 읽어 페이지 코드로 구현 (예: 놀거리 콜라주, 갤러리, 아티클, 제품 카드)
- 와이어프레임 PNG에서 **실제 이미지(히어로·식당·매장·갤러리)를 좌표 스캔 후 추출**하는 반복 작업
- HTML/CSS/JS **페이지 골격·인터랙션 코드 생성**, 헤드리스 브라우저로 **렌더 검증**
- 콜라보 비주얼·로고 모핑(오브 변형) 이미지 **생성 보조**

## 3. 본인이 직접 수정·결정한 부분
- **콜라보 컨셉**: "비비안 행성에 불시착한 이천 임금님표 쌀" 내러티브 기획
- **와이어프레임 전체**: 홈·About·Products·Icheon·Archive·Stores를 직접 디자인(Figma)하고 Bootstrap 그리드를 염두에 두고 구성
- **사이트 구조/페이지 구성**: 네비 메뉴와 각 페이지의 하위 섹션 구성 결정
- **페이지별 인터랙션을 직접 선택** (아래는 본인이 고른 결정들):
  - 홈 히어로: 여러 옵션 중 **자동재생 배경 영상형** 선택
  - 홈 Feature: 번호 클릭 시 이미지 교체(탭 전환식)
  - About 스토리보드: 스크롤 페이드인 선택
  - About 로고 모핑: 단순 흐름이 아니라 **"중앙에 딱 멈췄다 다시 이동하는 스냅"** 방식으로 구체화
  - 네비: 단순 강조가 아니라 **드롭다운(메가메뉴)** 으로 결정, About 하위=타임라인/심볼 지정
  - Products: 탭 즉시 교체 / 카드 호버 확대 / 하트=위시리스트 / Get now=구매 결정
  - 상품 상세: **쌀은 Size 대신 중량(4kg/10kg) 옵션**으로 분기 결정, ADD TO CART+Get now 2버튼 구성
  - Icheon: 맛집 클릭 시 지도 갱신, Discover 좌우 화살표=탭 전환, 핀 호버 강조, 도자기 항목 삭제 결정
  - Archive: 뉴스 카드 클릭→기사, 갤러리 화살표 전용+라이트박스+무한 루프 결정
  - Stores: 사진 클릭→지도, 호버 확대, 전화 tel 링크
  - 별도 페이지: 로그인=프론트 흉내+UI, 마이페이지=탭형+위시/장바구니 실동작, 구매=장바구니→주문서→완료+배송/환불 범위 결정
- **템플릿 리스킨 방향 결정**: 템플릿 기본 색/폰트를 **Pretendard + 흑백·타탄**으로 교체하도록 지시

## 4. 최종적으로 본인이 판단하고 결정한 부분
- AI가 제시한 인터랙션 옵션 중 **무엇을 채택할지 전부 본인이 선택**함
- AI가 정리한 명세서를 검토하고 **구조·범위를 확정**(예: 도자기 섹션 삭제, 인터뷰를 Archive에 귀속 등)

## 4.5 Bootstrap(Moderna) 템플릿을 어떻게 고쳤나 (핵심)

> 평가 포인트인 "기존 템플릿을 어떻게 활용·수정했는가"를 구체적으로 기록.

### (1) 수정 원칙 — 원본 보존 + 오버라이드 레이어
- 템플릿 원본 **`assets/css/style.css`는 한 줄도 수정하지 않음**. 대신 뒤에 로드되는 **`assets/css/custom.css`**를 새로 만들어 색·폰트·간격을 덮어씀(오버라이드). → 유지보수·복구 용이, "어디를 바꿨는지" 추적 가능.
- 템플릿 **`assets/js/main.js`도 무수정**. 우리 인터랙션은 **`assets/js/app.js`**를 따로 만들어 추가.

### (2) 리스킨 — 템플릿 기본 디자인을 우리 디자인으로 교체
| 항목 | 템플릿 원본 | 우리가 바꾼 것 |
|---|---|---|
| 메인 컬러 | 파란 계열(#1e4356 / #68A4C4) | **흑·백 미니멀(#101010)** + **레드 타탄**을 단일 컬러 포인트로 |
| 폰트 | Open Sans / Roboto (CDN) | **Pretendard** 로컬 폰트(@font-face, font/ 폴더 OTF 9종)로 전면 교체, CDN 제거 |
| 헤더 | 단색 파란 고정 헤더 | 홈=투명→스크롤 시 흰 배경, 서브=흰 배경 + 하단 보더로 상태 분기 |
| 푸터 | newsletter+링크 4열 | 멤버 2인 SNS + **하단 타탄 띠**로 개조, BootstrapMade 크레딧만 유지(무료 라이선스) |

### (3) 템플릿 컴포넌트 재사용 + 겉모습 교체
- **navbar `.dropdown`** → 메뉴명을 한국어/우리 페이지로 바꾸고 **호버 메가메뉴**로 확장(About·Products·Icheon·Archive 하위 항목)
- **AOS** → About 스토리보드·각 섹션 스크롤 페이드인 (단, 큰 이미지 늦은 로드로 reveal이 안 풀리던 문제를 **IntersectionObserver 안전망**으로 직접 보강)
- **Swiper** → Archive 갤러리 캐러셀(좌우 화살표·도트·무한 루프)
- **GLightbox** → 갤러리 이미지 클릭 확대
- **Bootstrap 그리드/컨테이너** → 전 페이지 반응형 레이아웃 골격
- **contact.html의 Google Map iframe** → Stores·Icheon 지도 연동으로 응용
- **back-to-top 버튼** → 흑백으로 리스킨해 유지

### (4) 템플릿에 없어서 직접 구현(신규)
- 홈 **영상 히어로/다크 영상 섹션**(YouTube 연동), 홈 **Feature 4탭 이미지 교체 JS**
- About **로고 모핑 스크롤 스냅 캐러셀**(휠→가로 전환, 중앙 강조 — 커스텀 JS, 별도 `symbol.html`)
- Icheon **놀거리 폴라로이드 콜라주**(타탄 배경), 맛집 **카드→지도 갱신**
- Products **옵션 분기**(쌀=중량) + **위시리스트/ADD TO CART 토스트**
- **장바구니 실연동**(수량·삭제·합계 실시간), **가입/로그인 프론트 흉내**(localStorage 계정 저장→로그인→네비 상태 전환)
